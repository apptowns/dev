# 갤럭시 워즈

## 게임 개요
### 게임 특징
- 킹덤 러시와 같은 타워 디펜스 게임
- 킹덤 러시의 우주판
- 킹덤러시 + 스타크래프트

### 세계관
- 고갈된 자원
  - 석유, 석탄은 물론이고 우라늄과 플루토늄마저 고갈된 지구는 새로운 자원을 찾아 우주로 눈을 돌린다. 
  - 오래지 않아 RF-340 행성에서 폴로드 개스를 발견하는데 이는 초전도체를 합성할수 있는 자원이었다.   
  - 채취를 위해 성간 우주선이 동원되고 엄청난 비용을 소모하기 때문에 정해진 적재량을 채우지 않으면 출발이 허용되지 않았다. 

- 행성의 주인
  - 행성에 인간들이 모여들자 행성 지하에 동면하고 있던 생명체가 깨어났다. 
  - 미네랄을 식량으로 하는 크립토들은 여러세기 전 소행성 충돌도 70%가 멸종하고 
  - 척박해진 환경 때문에 스스로 동면을 택한 것이었는데 트론의 첨단 기술로 플로드 개스가 체굴되자 
  - 이를 식량으로 하는 크립토들이 동면에서 깨어난 것이었다.  

- 미네랄을 건 사투
  - 이들은 마치 벌떼처럼 개스를 얻기 위해 트론을 공격하게 된다.
  - 하지만 막대한 부를 가져다줄 개스를 포기할수는 없었다.  
  - 행성 RF-340은 개스를 건 크립토와 트론의 전쟁터가 되고 만다.

### 채굴 기지를 지키기 위한 방어 시설 
- 체굴 기지를 지키기 위해 지형 지물을 이용해 타워를 건설해서 방어해야 한다.  
- 타워를 통과하여 채굴 기지를 공격하여 파괴되면 게임 종료 
- 체굴기지의 발열에 의해 크립토들도 체굴기지에 데미지를 입히고 죽게 된다.
- 채굴이 완료 됐으면 스테이지 클리어하고 지구로 귀환 한다.

  - 웨이브 : 적 웨이브

- 아이템
  - 우측 하단에 표기

  - 체력등을 볼수 있다. 

### 게임 구성 요소
- 최초 채굴 기지 안착 
  - 개스와 미네랄 채굴
  - 채굴이 완료되면 게임 종료
  - 몹을 클리어 해도 채굴 가능

- 체굴 기지
  - 거대한 우주선이며 일종의 화물선이다. 
  - 타워 관련 시설들을 실었을뿐 자체 방어 기능이 없다.
  - 시간당 체굴량이 있지만 해당 지역의 지형, 토질등에 따라 채굴이 완료 되는 시간이 다르다.
  - 체굴이 완료되면 바로 지구로 귀환한다.  
  - 남은 인원들은 이후 추가 우주선으로 RF-340 행성의 궤도 정거장으로 귀환한다.

- 포탑
  - 빠르게 건설해야 하므로 적정한 위치가 있어 아무데나 건설하지 못한다.
  - 궁수 타워 -> 마린 벙커 개념 (마린이 탑재 -> 거리와 속도로 업그레이드 -> 기본 머신건에서 최종 파괴력이 높은 무기로 교체) 
  - 병사 타워 -> 마린 병사(근거리 무기 사용 -> 파이어벳 참조) 3명 생산 (전투력 상승 -> 최종 로봇 강화슈트 입음)
  - 마법사 타워 -> 레이저 관련 무기 (관통 효과 -> 최종 관통및 속성 파괴)
  - 포탑 타워 -> 범위 데미지 (원작과 동일 디자인만 다름 -> 미사일?)

- 지원
  - 지원군 -> 우주기지에서 텔레포트해서 보내는 지원군 (병사 타워 군인보다는 마린에 가까움 기관총)
  - 마법 -> 위성무기로 대체 -> 불의 창이라 부름

- 추가 
  - 축복 -> 골드 버프 / 공격력 버프 / 뽑기 버프(확률 증가)
  - 영웅 -> 전쟁 영웅들을 따로 구할수 있음 (VIP 뽑기)
  - 아이템 -> 장착해서 1회성 사용 가능 (VIP 뽑기)

## 게임 세부 설명
### 스탯 정리


### 타워 (나무위키 참조)
1) 마린 벙커
- 벙커안에 머신건으로 무장한 마린이 있다. 
- 마린 1명,2명,3명으로 업그레이드
- 초월 업그레이드는 (공속이 떨어지고) 
  - 레인저 -> 특수탄을 사용하는 마린 -> 공중 공격 불가
  - 스나이퍼 -> 스플레시 데미지 + 장거리 가능 -> 공중 공격 불가   

2) 병사 기지 
- 근접 공격 마린
- 프라즈마 장갑을 낀 마린 3명 생산 (질럿가 비슷)
- 전투력 3단계 업그레이드
- 초월 업그레이드 (기능의 변화)
  - 파이어 뱃 -> 범위 공격
  - 바바리안 -> 로봇형 강화슈트 

3) 레이저 타워 
- 특수 공격 타워
- 속성 공격과 관통 공격등

4) 포탑  
- 범위공격과 물리데미지를 주는 포탄 발사 타워
- 던 높이 발사하여 더 강력한 파괴력 + 범위를 준다.
- 초월 업그레이드
  - 레이팜 : 폭탄의 범위가 2배 (포탄이 공중에서 터져 산개해서 내린다.)
  - 초고속 미사일 : 데미지가 2배 상공으로 높이 쏘아서 초고속으로 낙하한다.

### 지원
- 행성 괘도의 우주 정거장에서 지원 
- 위치에 따라 지원받지 못하는 지역이 있다. 
1) 지원군
- 정거장에서 양자 에너지로 텔레포트하여 특정위치로 전송 
- 외부 업그레이드 외에 인게임에서 업그레이드는 되지 않는다. 
- 2명이 한조이고 머신건으로 무장되어 있음 

2) 위성무기
- 특정 지역에 위성무기 발사
- 인게임 업그레이드 불가능하고 
- 불의 비라 불리우는 레이팜탄이 기본이다
- 아이템을 통해 1회성으로 다양한 무기를 사용할수 있다.

### 아이템
- 6종으로 구분된 18개의 아이템으로 
- 5개까지 장착이 가능하고 1회성으로 소모된다. 
  - 긴급수리 : 체굴기지가 공격 당할때 긴급 수리를 통해 30% 체력을 회복한다. 
  - TUB 개스 : 마린들이 휴대용을 가지고 있는 신경개스로 범위안에 일정 수준으로 움직임이 느려진다.
  - 소형 핵탄두 : 사용하면 화면안에 적들이 다 죽는다. 타워밖에 영웅을 제외한 모든 유닛들도 마찬가지
