# 슬라임 총을 잡다.
## 1. 설정
### 게임정보
- 쟝르 : 로그라이크 탄막 슈팅 게임
- 개발 기간 : 약 3개월
- 타겟 유저층 : 10 ~ 30대 (슈팅게임 유저)
- 세일즈포인트 : 화끈한 슈팅 게임..  

### 슬라임 총을 잡다!
- 캐릭터를 선택해서 6개의 무기까지 사용할 수 있으며
- 웨이브가 시작되면 정해진 시간까지 적을 처치하고 살아남아야 한다.
- 웨이브가 끝나면 레벨업에 대한 업그레이드와 상점에서 아이템을 구입할 수 있다.
- 20 웨이브까지 살아남는 것이 목적.
- 쉬움 < 평범 < 어려움 < 지옥 의 난이도가 있다.

### 세계관
1) 씨바 슬라임 됐어!
    - 뭔지 몰라도 내가 슬라임인건 알겠다.
    - 여기 저기 처맞고 다니고
    - 인간뿐 아니라 몬스터들도 나를 괴롭히네...
  
2) 웬지 모르는 전투의 기억들
    - 문득 문득 깨닫게 되는 무기를 다뤘던 기억 (전생의 나는 존윅?)
    - 무기를 다룰수 있게 된 슬라임 
  
3) 무기를 들고 우선 몬스터의 왕이 되자
    - 완전히 삐뚤어 질테다
    - 누가 말했던가 "먼저 몬스터의 왕이 되어라"
    - 슬라임 몬스터의 왕이 되기 위해 총을 들다
    - 오늘부터 삐뚤어질테다!

## 게임 설명
### 주요 기능
  - 자동 사격 무기가 기본으로 있고 동료 소환 버튼이 있다 (위급시 자신의 위치로 동료를 부른다.)
  - 스테이지 클리어 시간은 빠른 런 (30분 미만)
  - 상점에서 동료, 무기, 아이템을 강화 시킬수 있다. (인게임에서 적은 비용으로 강화된 아이템을 사용)
  - 동료, 아티팩트, 무기 모두 장착하면 드라마틱한 효과가 있지만
  - 단지 보유로 캐릭터의 기본 스탯이 올라간다.
  - 뽑기를 통해 스탯을 올리는 게임
  - 난이도가 올라갈수록 동료 / 아티팩트 / 무기에 추가 기능을 추가 할수 있다.
      - 추가 던전이 오픈된다. 

### 게임 진행 
1) 기본 메뉴
    - 캐릭터 : 캐릭터 선택
    - 카드 : 동료 / 아티팩트 / 무기 세팅     
    - 모험 : 스테이지 진입
    - 도전 : 맵을 추가 도전하면 카드에 추가 기능이 생긴다. 
    - 상점 : 카드 소환 패키지 구매등
 2) 세팅 :  로비 -> 캐릭터 선택 -> 무기, 아티팩트, 동료 세팅  
    - 캐릭터 선택
        - 3종 기본 제공 (플레이어 레벨을 통해 오픈)
        - 추가는 플레이를 통해 오픈된다. (보석구매 3종 / 이후 유료 구매) 
    - 카드 세팅
      - 무기, 아이펙트, 동료를 소환하여 세팅할수 있다.
      - 소환하여 수량으로 강화        

2) 인게임 : 스테이지 진입(난이도 선택) -> 웨이브 시작 -> 웨이브 클리어 -> 레벨업 (업그레이드와 아이템 구매)  
    - 난이도 선택 : 쉬움, 일반, 어려움, 지옥 (하위 난이도를 클리어 해야 상위 난이도 오픈)  
    - 스테이지 진입
      - 스테이지 한가운데서 시작
      - UI
        - HP : 적 데미지를 받아서 0이 돠면 죽는다.
        - AP : 방어력 게이지 (방패와 비슷한 개념 - 먼저 물리 데미지 흡수)
        - 레벨 : 캐릭터 레벨 -> 몹을 클리어 할때마다 추가 된다. (아웃 게임에서 캐릭터 선택 조건)
        - 웨이브 레벨과 시간 : 현재 웨이브 웨이브 레벨이 몇이고 몇초 남았는지
        - 보석 : 현재 보석량
        - 미션 : 퀘스트에 따라 지급되는 보석량이 표기
    - 웨이브 : 일정 시간동안 진행되는 적의 출현
      - 웨이브 종료시 상점에서 무기, 동료, 아이템을 구매 할수 있다.
      - 아웃게임에서 세팅을 잘 했다면 같은 가력으로 더 등급이 높은 무기, 동료, 아티펙트를 사용할수 있다. 
   
4) 스테이지 클리어 : 반복되는 웨이브와 레벨업 -> 20 웨이브까지 생존이 목표
    - 위와 같은 웨이브와 웨이브 클리어 그에 따른 레벨어을 통해 상점에서 아이템 구매가 반복되며 캐릭터가 성장한다.
    - 이렇게 20스테이지까지 클리어 하고 생존하면 스테이지 클리어
    - 스테이지를 클리어 하면 이후 상위 등급 난이도를 플레이 할수 있다. 
  
  5) 보석 보상
    - 게임내 이벤트를 통해 보석을 보상한다.
    - 보석은 아웃 게임에서 중요 재화이다.

## 게임 개발 관련 시방서 
### 1) 타이틀 씬
  - 버전체크
    - 게임 버전 : 같으면 소스 버전 체크로 이동 틀리면 새 클라이언트 다운로드를 위해 스토어로 이동
    - 소스 버전 : 같으면 로컬에서 데이터를 찾고 틀리면 서버에서 다운로드
  - 데이터 저장
    - 로컬에 데이터를 저장한다. (파싱하지 않는다.)
  - 로그인
    - 최초 진입시 하단에 약관동의 버튼을 누르면 로그인 창이 뜸
    - 구글 로그인
    - 애플 로그인
    - 게스트 로그인
    - 이미 로그인 했다면 터치 투 스타트 메세지 
  - 기타
    - 사운드 옵션 (효과음, 배경음, 보이스)
    - 공지사항  

### 2) 로딩
  - 페이드인 아웃으로 진행
  - 게임 데이터 파싱

### 3) 로비
  - 기본 메뉴 : 로비 하단에 표기
    - 캐릭터 : 3종 기본 제공
      - 수련 : 골드로 스탯을 상승 
      - 추가 캐릭터는 구매 조건이 완성되면 추가 3종 외에 유료로 구매한다. 
    - 무기
      - 무기 강화 화면
      - 공격력에 영향
      - VIP 뽑기로 강화 가능 - 개별및 일괄 강화 가능
    - 동료
      - 같이 다니는 AI 캐릭터로 인게임에서 플레이 주면으로 부를수 있다. 
      - 동료는 1가지 무기만 소지 한고 체력이 소모되어 죽게 되면 리젠시간 후에 플레이어 옆에 다시 등장한다.
      - 스탯은 플레이어의 n% 로 성정되고 최대 레벨은 캐릭터마다 다르다. (부분적으로 200% 가능)
      - VIP 뽑기로 강화 가능 - 개별및 일괄 강화 가능  
    - 아이템
      - 캐릭의 스탯을 강화 할수 있다.
      - 
    -  상점
      - 축복, 광고제거, 소환, 보석구매, 패키지 
    - 모험
      - 스테이지 진행
      - 클릭하면 난이도 설정 화면이 나온다.
      - 이후 게임 진행  
  - 기타 메뉴



### 2) 스테이지 진입
- 스테이지 진입 (18개 스테이지)
- 각 스테이지마다 웨이브가 설정 (30 ~ 60)
- 10 웨이브마다 체력이 높은 보스 등장
- 최종 웨이브에 보스 등장

### 3) 전투 - 웨이브와 레벨업
- 웨이브 
  - 일정 수량의 몬스터를 물리치면 웨이브 클리어
  - 웨이브 총량 -> 수식을 통해 웨이브 총량을 제어
  - 개별 몹 체력 -> 상,중,하로 구분
    - 중 -> 웨이브 총량에 n / 1 (몹의 수 결정)
    - 상 -> 중 x 3  or 중 (마법) (수만큼 결정된 수에서 x3 차감)
    - 하 -> 중 / 3 (중의 수량에서 n%로 차감후에...)

- 슬라임 총을 잡다.
  - 무장 -> 무기 최대 3 ~ 9종 동시 사용가능 
  - 패시브 스킬 -> 최대 6종 사용가능
  - 동료 -> 6명까지 AI 전투 지원
    - 동료는 하나의 무기만 사용
    - 최대 6명까지  

- 보상
  - 몹을 클리어 하면 보상 
    - 골드 -> 골드로 경험치 축적
    - 보석 -> 보석으로 VIP 뽑기 (업적으로도 추가 지급 가능)
    - 아이템 -> HP, MP 물병
       
- 보스
  - 중간보스 - 10 웨이브마다 출현 (중 x 10)
  - 최종보스 - 스테이지 최종 웨이브에 출현 (중 x 100)

- 상자 (아래 중 하나 획득)
  - 무기 - 패시브용 무기
  - 장비 - 공격력 or 방어력 상승
  - 보급 - 음식 (체력과 공격력 상승)
  - 카드 - 스탯 상승

- 상점 (레벨업시에 사용 가능)
  - 스테이지에서 획득한 보석으로 구매 가능
  - 액티브 스킬
  - 패시브 스킬 

- 무장 
  - 웨이브 클리어시에 상점에서 구매
  - 무기류및 패시브 스킬 구매 가능 (돈이 있으면 다 구매 가능)
  - 최대 3 ~ 9 종의 무기 소유 가능 

### 4) BM
- 축복 -> 광고보고 축복 (골드 / 체력 / 공격력) -> 유료 상품으로 구매 가능
- 보석으로 VIP 뽑기 -> 무기, 스킬, 동료 
- 무장 슬라임 -> 기본 슬라임은 보유효과로 능력을 취하고 새 슬라임은 장착 스킬 추가 
  - 전설의 궁수 슬라임 : 시야 확장, 원거리 사거리 150% 증가, 스피드업 30%
  - 광전사 슬라임 : 일정 확률로 광전사 폭주 -> 공격 스탯 모두 상승, 체력 반감
- 특정 무기, 스킬, 동료 구매
- 패키지 구매
