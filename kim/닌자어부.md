## 액션
- 방향성있음 / 우측이 기본 / 32컷 1초
### 1) 아이들
- idle 0 (1초) : 꼬리를 내리고 숨쉬는 동작
- idle 1 (1초): 같은 동작에서 귀를 움직인다 (2~4에 한번씩 랜덤하게 반응)
### 2) 이동
- move 0 (2초): 걷기
- move 1 (1초): 낙법으로 이동
### 3) 어택
- attack 0 (1초): 인술1 -> 꼬리를 세운 상태로 귀를 쫑긋하고 가슴에 손을 올린다. -> 기본 0
- attack 1 (1초): 인술2 -> 같은 동작을 점프해서 한다.-> 특정 상황에서 발생
### 4) 승리
- win (3초): 팔짱을 끼고 좌우를 보며 으시대는 모습 
### 5) 미스
- damage (2초) : 뒤로 자빠진다.
