## 1. 게임 작품명 
1. 제목 : 비트탈고양
2. 의도
    - 비트 게임임을 강조
    - 고양이가 주인공인 고양이 게임임을 강조
    - 고양이소리를 기반으로 한 제목 제작
    - 영문명 beatcat
 
## 2. 게임 개요

1. 게임 개요
    - 세계 최초로 시도되는 리듬퍼즐 게임이다.

2. 기획의도.
    - 퍼즐게임의 사고력, 리듬게임의 순발력을 조합하여
    - 누구나 가볍게 즐길수 있는 신쟝르
    - 리듬퍼즐 캐주얼 게임 제작 목표    

3. 타겟 이둉층.
    - 10대 20대 여성을 중심으로 한 라이트한 유저 

4. 플랫폼 소개
    - 모바일 환경 (IOS, AOS)
    - 스팀 (유료 판매 예정으로 모바일과 BM이 다르다.)

5. 게임 쟝르
    - 세시 소프트가 제안하는 게임은 리듬퍼즐 게임이다.
    - 참고 게임
        - 네크로댄서 (리듬 액션 게임)
            - 리듬에 맞춰 액션을 하는 리듬 액션 게임
            - 이러한 쟝르 게임의 효시가 되었다. 
        - 룸오브데스 (액션 퍼즐 게임)
            - 움직임을 제한 하는 퍼즐 게임
            - 창의 움직임을 제한하여 던전의 이동을 푸는 퍼즐 게임

6. 그래픽 컨셉
    - 16픽셀을 한 폰트로 하는 저도트 게임
    - 제작 방식
        - 도트 그래픽 게임
        - 16픽셀을 기본 폰트로 하는 타일 제작 방식 게임
        - 16픽셀 기본이기 때문에 저도트 게임군에 들어간다.
            - 장점 : 표현의 한계 때문에 제작사 규모에 따른 제작 비용의 차이가 적다.
            - 단점 : 같은 이유로 표현의 한계로 숙력 디자이너 상다수 필요 (저예산이이도 최저 예산이 적지 않다.) 
        

7. 플레이 방식
    - 비트 게임 시스템
        - 비트에 따라 커맨드를 입력해야 한다. (비트가 맞지 않으면 조작이 무시된다.)
        - 모든 기믹과 몹은 비트에 따라 움직인다.
        - 비트에 따라 "무궁화 꽃이 피었습니다라"는 게임을 하는 느낌
        - 일명 무궁화시스템이라고 부른다.
    - 회전 퍼즐 시스템
        - 더전은 좁고 칼을 무겁고 크다.
        - 적절한 공간을 확보하여 회전하여야 이동이 가능하다.
        - 공간이 막히면 이동 자체가 불가능 할수도 있다. 
    - 진행 방식
        - 캔디크러시 사가처럼 무한 레벨 시스템
        - 레벨을 클리어 하면 다음 레벨을 진행 할수 있다.
        - 진행한 만큼 세이브되어 언제던 재진행 할수 있다.  
        - 오픈스펙 (200레벨로 오픈)
            - NDM 시스템으로 실시간으로 레벨 추가가 가능하다.
            - NDM?
                - 네트워크 던전 메이커로 구글 시트로 맵을 구성하면
                - 게임 시작시 해당 시트 데이터로 맵을 자동 구성하는 툴
                - 유니티의 기본 번들 개념이 아니라 실시간 레벨 추가가 가능하다.      

## 3. 게임아이디어 기획 배경 및 특징
1. 기획 배경.  
    - 발단 => 신쟝르의 목격
        - 네크로댄서의 유니크함
        - 리듬 게임의 또다른 발전 가능성에 주목

    - 전개 => 이종쟝르의 조합에 기획 초점
        - 리듬 게임이 기본이 되며 타장르의 조합은 없을까?
        - 조합이 될 쟝르에 대한 기획 고민
        - 던전형 퍼즐 게임에 주목 (네크로댄서를 참고)    
    
    - 위기 => 이질적 쟝르의 부조화
        - 순발력이 필요한 리듬 게임과 사고력이 필요한 퍼즐의 부조화
        - 재미는 있으나 뭔가 어색하다.
        - 옳았는가에 대한 고민
    
    - 절정 => 난이도에 대한 고민
        - 퍼즐 난이도가 높으면 리듬 게임이 불편하게 느껴진다.
        - 퍼즐 부분의 난이도가 낮아도 리듬이 접합되어 충분히 재밌다는 결론
        - 그러나 팀원들끼리 게임성에 대한 이견이 가라앉지 않아 게임성 테스트
    
    - 결말 => 개발 2달만에 레벨 30개로 구글 스토어에 테스트 버전 출시
        - 어려우면 생각보다 많은 사람이 포기하게 된다.
        - 어려우면 이탈률이 높아서 유저가 재미를 느낄 기회조차 없다는 것을 인지
        - 난이도를 하향하는 것으로 최종 결정
        - 스테이지 방식에서 레벨 방식으로 전환도 함께 결정 (재 게임시 이탈 발생)     

2. 게임 특징
    - 유니크 하다.
        - 일단 전세계 어디도 없는 쟝르
        - RPG의 외피를 입었지만 리듬퍼즐게임
    - 그러나 재밌다.
        - 퍼즐을 푸는 재미
        - 리듬게임으로 난관을 이겨내는 재미가 있다.
         
## 4. 시높시스
1. 8왕국아르는 나라
    - 8왕국이 배경
    - 8개의 왕국이 모여 연맹을 이루며 연맹이 추대한 황제가 있다.
    - 각각의 왕국은 인간, 드워프, 오크, 엘프등이 혼합하거나 독자적으로 구성되어 있다. 

2. 미움받은 황제의 아들
    - 주인공 판은 8황국 황제 다리우스 13세의 아들이다.
    - 하나뿐인 핏줄이지만 황제가 왕자를 미워해 북쪽탑이라 불리우는 헤르듬 감옥에 갇혔다.

3. 반란으로 황제가 유폐되었다. 
    - 어느날 헤르듬에 황제의 측신인 에르두가 찾아와 황제의 보검을 전해준다.
    - 바르 공작이 반란을 일으켰고 황제는 유폐됐다는 소식
    - 판은 에르두의 딸 앨리스의 안부를 묻는다.
    - 앨리스는 판의 소꼽친구 둘은 한쌍의 고양이라고 불리웠다.

4. 판 한쌍의 고양이 앨리스를 구하러 나선다. 
    - 감옥이 열리고 게임 시작
    - 게임이 시작되면 판은 고양이로 변신 이는 앨리스를 만나 모든 문제가 해결될때까지 지속된다.
    - 판은 앨리스만 생각한다는 뜻     

## 5. 등장 캐릭터및 몬스터 설정
1. 캐릭터 설정   
    - 능력치 설정
        - 사정거리 : 16x16을 한 타일 크기와 거리를 계산한다. 
        - 공격력 : 무기에 따라 1~5까지 존재 2가 넘어가면 양손 사용으로 방패를 사용할수 없다.
        - 방어력 : 방패를 사용하면 존재하며 물리 방어력은 1로 고정되어 있고
                    - 얼음, 독, 번개를 방어하는 방패가 따로 존재한다. 
            
        - 생명 : 사실상 입장권으로 기본 1개로 고정되며 하트로 표시된다.
                  - 하트가 2개인것은 게임오버가 되도 연속으로 한번 더 할수 있다는 의미다.
                  - 30, 60, 90... 레벨 클리어 할때마다 해당 스테이지 보스가 추가 하트를 준다.
                  - 레이드를 하게 되면 현재 하트를 추가 충전할수 없기 때문에 하트가 몇개인지가 중요하다.   
        - 능력 : 아이템을 통해 부여 받는 능력으로 20, 40, 60.. 단위 보스들에게 받는다.
                - 미는 능력 -> 타이탄의 장갑을 끼면 생긴다.
                - 활 사용 -> 오딘의 활을 얻으면 생긴다. 방향만 맞으면 적중률 100%
                - 그외 불 붙이기, 부메랑 던지기, 점프, 헤엄치기등 모두 아이템을 얻으면 얻어지는 능력이다.           
           

2. 캐릭터 설명 - 인게임 등장 캐릭터
    - 판
        - 설정 : 주인공이자 플레이어 캐릭터
        - 사정거리 : 2타일
        - 공격력 : 무기에 따라 달라진다. 기존 1에서 시작
        - 방어력 : 방패에 따라 달라진다. 기본 1에서 시작  
    - 디펜서 방패 스켈레톤
        - 등장위치 : 헤르듬 감옥
        - 설정 : 방패를 들고 던전에서 위치를 지킨다.
        - 사정거리 : 1타일
        - 공격력 / 공격력 : 0으로 무기를 들고 있지 않다. / 1  

    - 디펜서 장님 스켈레톤
        - 등장위치 : 헤르듬 감옥
        - 설정 : 앞을 보지 못해 던전에 박혀 있어서 움직일수 없다. 디펜서와 오펜서의 특징을 함께 보유
        - 사정거리 : 1타일
        - 공격력 / 방어력 : 1 / 1
         
    - 오펜서 간수
        - 설정 : 원래 헤르듬의 간수였다.
        - 사정거리 : 1타일
        - 공격력 / 방어력 : 1 / 1         
     - 오펜서 쇠공 스켈레톤
        - 설정 : 쇠공을 던지는 스켈레톤으로 공격시 쇠공은 철퇴로 변한다.
        - 사정거리 : 2타일
        - 공격력 / 방어력 : 1 / 1   
3. 캐릭터 설명 - 스토리 연관 캐릭터
    - 판
        - 미움받는 황제의 아들로 황제에 의해 감옥에 유폐되었다.
        - 반란의 이야기를 듣고 자신의 친구 앨리스를 걱정하는 판
    - 앨리스
        - 측신 에르두의 딸이며 황제의 시종으로 있다.
        - 반란으로 황제와 함께 유폐되었다.
    - 황제 다리우스 13세
        - 8왕국의 황제로 동생의 남편이며 사실상의 왕국의 최고 실세인 바르 공작의 야심을 알고 있다.
        - 바르에게서 판을 보호하려 마움에 눈이 먼것처럼 행동하며 판을 감옥에 가두게 된다.
        - 모든 문제가 해결되고도 부자사이의 관계는 회복되지 않는다.

    - 바르 공작
        - 다리우스의 동생과 결혼하여 왕위 서열 4위(부인이 3위)이다.
        - 판이 유폐되었지만 다리우스보다 나이가 많았기에
        - 결국 본인이 먼저 죽을거 같아 반란을 일으켰다. (삼국사기 고구려 본기 내용 차용)
        - 끔찍하게 아끼는 아들이 있다. 그의 자식사랑은 늘 황제와 비교 되었다.   
    - 엘레나 공주
        - 다리우스의 여동생 왕위 서열 3위로 바르보다 욕심이 더 많은 표독스러운 악녀
        - 절세 미녀로 유명하고 많은 나이임에도 불구하고 너무 젊어 보여 왕국에 많은 소문이 있다.
        - 언데드와 관련 있는 인물  

    - 이스마엘 왕자 
        - 바르와 엘레나의 아들이며 판의 친구로 부모에 비해 너무 착한 아들
        - 판과 어릴때부터 비교 되었지만 늘 판에게 양보하였다.
        - 앨리스와 판과 삼총사였으나 게임에서 이스마엘은 고양이로 나타나지 않는다.
        - 언데드 왕 레오나드 바라막의 핏줄이며 결국 각성하여 자신이 좋아하는 앨리스 앞에서 언데드가 된다. 

    - 왕 레오나드
        - 전능한 왕 레오나드라 불리운다.
        - 엘레나를 사랑하였으나 권력이 약했던 당시의 레오나드는 인간인 그녀와 결혼하지 못했다.
        - 엘레나가 결혼하고도 밤의 노랫소리로 변신하여 그녀를 만났고 미래를 기약하기 위해
        - 이스마엘인 자신의 자식임을 알자 폭주한다.
  
6. 인터페이스
    - 타이틀 화면
        - 언어 선택 : 한,영,일,중,독,프,스,이 8개국 언어 제공
        - 버전 코드 : 현재 버전의 코드
        - 공지 : 중요 공지 -> 대부분 광고를 보여 준다.
        - 라운지 연결
        - 최초
            - 닉네임 설정 : 닉네임을 설정 할수 있다.
        - 재방문
            - 클래식 모드 : 스테이지가 제공 난이도 1개에서 4개까지 수준의 스테이지 제공
            - 레이드 모드 : 스토리에 따라 진행 게임 종료후 턴을 기록하여 순위를 정하여 랭킹으로 표시된다. 
    - 로비 화면
        - 스테이지
            - 스테이지 난이도 표기 : 1~4개
            - 스테이지 표기     
        - 업적
            - 스테이지를 클리어 할 경우 재 방문하면 추가 보상이 없다.
            - 이 경우 각종 업적으로 코인을 보상을 받아야 한다.
            - 디펜서 n마리 처치 (5마리 단위로 코인 1개 보상)
            - 오펜서 n마리 처치 (5마리 단위로 코인 1개 보상)
            - 아이템 n개 획득 (개당 코인 2개 보상)
            - 접속 n초 보상 (100초에 코인 1개 보상)
            - 고급 접속시간 보상 (1000초에 코인 10개 보상 - 중복 보상)
            - ...  
        - 상점
            - 무료 충전 : 코인으로 하트를 무료 충전 한다.
            - 무기 / 방패를 코인으로 구매
            - 그외 패키지 구매
            - 광고 제거등 
        - 도감
            - 기믹, 몹등을 표기

        - 하트 : 1회 피격 당하면 사망 (보스 클리어시마다 최대 하트가 1개씩 늘어난다.)

        - 비트 체크 : 판에게 닿을때 커맨드를 입력하면 조작이 가능하다.

        - 공격키 : 공격버튼

9. 게임 시스템 설정
    - 스테이지
        - 하트를 소모하여 게임에 진입한다. (하트는 사실상 입장권)
        - 캔디크러시사가처럼 1 스테이지를 클리어 하면 2스테이지를 플레이 할수 있다. (23년 9월 1일 오픈테스트후 시스템 수정)
        - 30의 배수마다 보스판이 있고 클리어하면 최대 하트 수가 늘어난다.
        - 20의 배수마다 중간보스가 존재하고 클리어하면 아이템을 준다.
        - 하트가 있다면 계속 재 게임을 할수 있다. 
    
    - 스테이지 클리어  보상
        - 각 스테이지는 난이도가 평가되어 클리어 하면 1개에서 최대 4의 코인을 받을수 있다.
        - 리듬 판정이 높을 경우 추가 코인 1개, 시간 타임이 50%안에 들었을 경우 추가 1개 코인을 준다.
        - 추가 보상을 포함하면 최대 코인수는 3~6개가 된다.  
     
    - 스테이지 컨셉
        - 클래식모드와 레이드 모드가 있다.
        - 레이드는 스토리에 맞는 스테이지가 나열된다.
            - 헤가듬 감옥 -> 도적들의 요새 -> 바라안 숲 -> 고든의 저택 지하... 
        - 클래식 모드는 제작사가 배치하는 순서로 스테이지가 배치된다.

    - 스테이지 응모 시스템
        - 제작사가 안내하는 방법으로 시트를 제작하면 맵을 테스트 해볼수 있다.
        - 이를 제작사에 응모하면 심사후 클래식 모드에 추가 된다.
        - 추가된 맵을 유저가 플레이 할 경우 음모자는 1회 플레이에 코인 1개를 받는다.
        - 이 코인은 장비를 구입할때 사용할수 있다.
