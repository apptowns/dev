## 게임 개요.
### 개요
- 2023년판 스페이스 오페라 
- 방치형 게임 
- 제목 : 스페이스 크래프트 : 벌레의 습격
  
### 배경 스토리
1) 서기 2084년 tp - 45 행성에 엄청난량의 미네날과 가스의 매장이 확인 되었다.
    - 미네랄과 개스를 자원으로 하는 인간은 행성에 대규모 자원 채굴단을 급파 하였다. 
    - 행성 각 지역에 자원을 체취하기 위해 우주선형 이동기지들이 내려오고 
    - 인간은 자원의 원활한 채취를 위해 -139도의 환경을 인위적으로 영상에 가깝게 조절하였다. 

2) 기온이 올라가자 지하 깊숙한 곳에서 동면하던 벌레들이 깨어났다. 
    - 바뀐 환경으로 지하 깊숙한 곳에서 3만년동안 동면해 있던 벌레들이 깨어났다.
    - 빙하기에 스스로 동면을 택한 벌레들이었다.  
    - 처음엔 큰 위협이 되지 않던 벌레들은 점점 강력해지고 
    - 뒤늦게 환경을 돌리려고 해도 벌레들은 이번에는 동면 대신 환경에 적응을 택하였다. 

3) 고갈되어 가고 있는 자원인 미네랄과 개스, 목숨을 걸고 채굴 할수 밖에 없다. 
    - 전 우주적으로 고갈되어 가는 미네랄과 가스를 포기하기란 쉬운일이 아니었다. 
    - 자원 채굴을 위해 인간은 결국 벌레와의 전쟁을 하게 되는데...   

### 참고 설정
1) 이동기지
    - 우주선으로 사용 가능한 기지
    - 착륙하면 기지
    - 거대한 기가레이저로 무장
    - 내부 대부분을 미네랄과 개스로 채우는 일종의 화물선이다. 

2) 위성 무기
    - 행성 주변에 배치된 64개의 공격형 위성을 통해 사용
    - 4개의 초고속 폭탄(빠른 속도로 하강하여 소형 핵에 준하는 범위 포격을 한다.)
    - 레이저, 신경 폭탄등 사용가능

3) 유닛 특징
    - 스타크래프트의 테란, 저그, 프로토스의 특징을 가지고 있다. 
    - 벌레들의 특징 -> 저그와 동일하다. 

### 기획의도
1) 스페이스 판타지 판 방치형 디펜스 게임
2) 모바일 형 뽑기를 통한 자동 스타크래프트 
3) vip 뽑기 + 시즌 패스를 통한 유료 모델 게임

### 벤치 마킹 게임
1) 더 타워 아이들 타워 디펜스 -> 기본 게임 시스템 차용
    - 중앙에 기지가 있고 사정거리안에 몹이 들어오면 처지한다. 
    - 몹을 처지하고 얻어진 재화를 통해 기지의 능력을 강화한다.
    - 레벨업을 통해 오브를 생성하여 기지를 외곽에서 방어한다.
      - 본 게임의 기지는 마차로
      - 오브는 동료로 차용        

2) 스타크래프트
    - 테란 설정이 배경 -> 이동하는 메인기지가 주인공
    - 부대는 테란, 저그, 프로토스 유닛중 물리적 공격력 위주로 구성 
    - 핵을 위성무기로 데체하고 메인 스킬로 사용
    - 기지가 성장하여 뽑기를 통해 부대를 뽑아 전투하는 개념

3)  레전드 오브 슬라임 -> 스테이지 구성과 VIP 뽑기
    - 스테이지 구성 (방치형 게임의 일반적인 구성)
        - 해당 진행 구간을 클리어 하지 못하면 반복 진행 -> 골드 확보를 통해 강화하여 클리어
        - 보스클 클리어 하면 경험치 대폭 상승 
    - VIP 뽑기 
        - 뽑기 횟수가 많아지면 뽑기 레벨이 올라가 상위 등급 확률이 높아진다.  
    - 시즌패스
        - 참고하여 추후 기획

### 게임 구조
1) 미네랄과 개스를 채취해가기 위해 벌레에게서 기지를 방어해야 한다.
    - 미네랄과 가스가 매장된 곳에 이동기지를 내린다. 
    - 같은 이유로 미네랄과 개스를 노리는 벌레들의 공격을 방어해야 한다. 
    - 부대가 구성되기 전에는 이동기지의 메인 레이저에 의지해서 방어해야 한다. 
    - 강화를 통해 부대 생성이 가능하게 성장해야 한다. 
2) 기지포
    - 기지에 장착되어 있는 기본 무기 
    - 사정거리안에 모든 적 유닛을 파괴한다.
    - 외형은 동일하나 성능을 개선할수 있고 이는 뽑기를 통해 가능하다.
    - 총 24으로 수직 서열화 되어 있다. 

3) 뽑기를 통해 부대를 양성하여 기지를 보호해야 한다.
    - 부대를 구성하는 유닛은 사용 제한이 있다. (기지 레벨)
    - 부대의 인원 제한이 있다. (강화를 통해 해결)
    - 뽑기에는 제한이 없다. (제한 없음 - 위의 두조건에 따른 사용 제한만 있음) 
      
4) 위성 무기을 통해 강력한 무기를 사용할수 있다. 
    - 경량 핵 -> 비상 상황일 경우 사용, 아군 유닛이 파괴되지만 수리 할수 있다. 적군 괘멸
    - 레이저 -> 사정거리안에 가장 큰 몹을 집중 타격
    - 신경파 -> 사정거리안에 적군 멈춤
    - 미네랄 투척 -> 미네랄을 먹이로 하는 벌레들이 서로를 공격한다.

### 부대
1) 부래란?
    - 기지를 보호하는 유닛들 
2) 부대의 구성
    - 특성에 따라 구분 -> 6계열로 구분
    - 희귀도에 따라 노말, 고급, 레어, 영웅, 전설, 신화 6등급으로 구분된다. 
    - 희귀도 등급안에서 다시 5등급으로 나뉜다.

3) 강화
    - 강화를 통해 능력이 올라가고 
    - 스킬이 늘어난다. 
    - 유닛의 스킬은 액티브 스킬이지만 시간에 따라 자동으로 충전되어 사용된다.
    - 각 유닛마다 3종  
    
4) 융합
    - 봅기를 통해 동일한 유닛이 나오면 융합하여 5 -> 1개로 만들수 있고 
    - 융합한 유닛이 5개가 보이면 상위 등급으로 승급 가능하다.
    - 커세어 25개를 뽑으면 레이스로 승급 할수 있다. 
    - 레이스 25개를 봅으면 스카우트로 승급 할수 있다.
    

### 게임 관련 언어 정리
1) 배경 관련 
    - 웨이브 : 일정시간 몰려오는 적의 패턴 보통 10초 ~ 5분.
    - 레벨 : 웨이브 4번을 클리어 하면 보스가 나오고 이 보스를 클리어 하면 레벨을 클리어 했다고 설정
    - 에피소드 : 10개의 레벨을 클리어 하면 에피소드 1개가 클리어 된걸로 설정 
    - 스테이지 : 에피소드를 모으면 하나의 스테이지가 된다. (스테이지마다 에피소드의 수량이 다르다.)

2) VIP 뽑기란?
    - 가차풀이 부대 유닛 30명, 무기 24종으로 고정되어 있다. 
    - 무기와 동료 각각 수직으로 등급화 되어 있다. 
    - 뽑기 레벨이 올라갈수록 상위 등급을 뽑을 확률이 높아진다.

3) 강화
    - 기지 강화 : 골드로 강화 -> 능력 향상
    - 부대 강화 : 골드로 강화 -> 유닛 능력 향상, 스킬 습득
    - 위성무기 강화 : 골드로 강화 -> 연구 개발을 통해 위성 무기 강화

4) 아카데미 
    - 전체 유닛의 새로운 기술을 연구한다.  
    - 연구는 시간으로 측정된다. -> 레벨이 올라갈수록 점점 긴 시간을 소모한다. 
    - 연구는 유료 재화인 다이아로 즉시 완료가 가능하다.
    - 연구 항목
      - 위성무기 능력 연구
      - 부대 유닛의 스킬 능력 연구
      - 기지 관련 연구 
      - 무기 성능 연구
     
5) 컨디션
    - 낯 -> 밤 난이도 상승
    - 맑은 날씨 -> 흐린 날씨 난이도 상승
    - 온도가 낮아지면 전반적으로 몹의 컨디션이 올라간다.
    - 동면기가 길었던 만큼 다음 추위는 확력을 올려 공격력을 상승 시킨다.
    - 대신 체력을 떨어지는 편

6) 이벤트
    - 스토리 관련 이벤트

## UI 설명
### 인게임
1) 레벨 : 포장마차 (주인공) 레벨
2) 날씨 : 맑음 -> 흐림(비)까지 날씨 변화 (국지적 난이도 상승 요소)
3) 골드 : 현재 최대 골드량
4) 보석 : 현재 최대 다이아량
5) 생존시간 : 일반 게임 스테이지으 다른 표현 
6) 동료 : 포장 마차 주변을 방어하는 동료 (최대 4명)

### 하단부
1) 좌측 부터 강화, 무기, 동료, 도전, 샵
2) 강화 -> 마차로 표기되는 주인공의 스탯 
3) 무기 -> 마차에 장착되는 24등급으 머신건 관련 UI
4) 동료 -> 마차를 호위하는 동료 최대 6명 (마차 외부 4명, 내부 2명)
5) 스테이지 -> 기획중 -> 현재는 일반 방치형 게임처럼 진행 할 예장
6) 도전 -> 특수 목적 던전 같은 개념 -> 보스도전, 골드 도전, 무한도전
7) 샵 -> 패키지, 시즌패스, VIP 뽑기(무기, 동료) 다이아, 골드 판매

## 게임 진행
### 게임 시작
1) 강화를 통해 전체 스탯을 올린다.
    - 마을 한가운데 마차를 세워놓고 시작
    - 캐릭터 즉, 대장인 나를 강화한다.
    - 능력 강화는 전체 무기, 동료들의 스킬에 기반이 된다.
    - [강화 스탯 정리](#강화-스탯-정리) 

2) 무기를 장착하고 강화와 융합을 통해 공격력을 올린다.
    - 마차에 탑승한 주인공의 무기로 기관총이다. 
    - 무기를 수리(강화) 하여 능력을 강화 한다.
    - 무기를 교체(승급)하여 능력을 상승 시킨다.
    - [무기를 구분하는 스탯](#무기-구분-스탯) 
    - [무기의 능력을 구분하는 스탯](#무기-능력-스탯)   

3) 동료
    - 마을 사람들이 마을을 수호하는 나를 따르게 된다.
    - 뽑기를 통해 소환하는 것을 고용이라고 한다.
    - 고용된 동료를 

4) 스킬
 
5) 몹을 물리치고 마을을 방어하면 스테이지를 클리어 할수 있다.
    - 스테이지 구성      
      - 웨이브 4개 -> 레벨 -> 레벨마다 보스 존재
      - 레벨 10 -> 에피소드 1개
      - 에피소드 24개 -> 스테이지로 구성되어 있다.    
    - 스테이지의 클리어
      - 보스를 물리치고 레벨을 클리어하면 다음 에피소드가 진행된다.
      - 최초 오전 10시부터 다음날 10시까지가 진행시간이다. (게임 내 시간) 
      - 시간이 흐를수록 날씨가 나빠질스럭 난이도가 올라간다.    

6) 스테이지 클리어가 힘들 경우 추가 스탯의 강화하여 능력을 강화 한다. 
    - 스테이지 진행 됨에 따라 차례로 추가 스탯들이 개방된다. (캐릭터 레벨도 고민중) 
    - 공격 스피드부터 개방된다.
    - [강화 스탯 정리](#강화-스탯-정리) 
 
7)몹의 종류
    - 마차를 공격하는 생명체(?)
    - 야수, 인간, 마차등 서부 판타지의 많은 
    - 야수 -> 늑대, 코요태, 멧돼지등 맹수에거 쥐등 농부에게 해가 되는 짐승들
    - 사람 -> 부랑자, 강도, 무법자, 탈영병, 악한 인디언등 
    - 기물 -> 마차나 기관총 부태등 야수나 사람에 속하지 않는 몹을 통칭
    - 좀 더 크게 표현 된 몹들이 해당 레벨의 보스로 나타난다.
   

8) 이벤트 진행 
    - 화면 상단 우측에 이벤트 표기 ㄱ4(경험치 획득 기준에 따라 이벤트 진행)
    - [이벤트 리스트](#이벤트-리스트)      
    - 다이아로 이벤트 보상 -> 자연스럽게 VIP 뽑기로 유도
    - 스테이지 이벤트 진행 => 스테이지로 이동 (기획중)
    - 몹도 기본 스탯에 영향을 받는다. 동료를 구성하는 것과 같은 형태로 몹도 구성된다.
    - [몹 관련 스탯 정리](#몹-관련-스탯) 

9) VIP 뽑기
    - 게임 진행을 위해 스킬, 무기, 동료는 스탯 상승의 플러스 요인
    - 무기 -> 물리적 공겨력 (사거리 만큼 기지 방어)
    - 동료는 -> 물리적 공격력 (무기의 사거리 밖에서 기지 방어)
    - 스킬 -> 
    - 그런데 이 세가지는 모두 VIP 뽑기를 통해서만 습득 할수 있다. 
    - 게임은 다이아를 지속적으로 공급해주고 뽑기로 유저를 유저하는 구조를 가지고 있다.

## 게임 스테이터스
### 게임 데이터 스탯
  - 강화되지 않으나 게임 진행을 위해 필요한 스탯 정리
  
### 강화 스탯 정리
  - 주인공 스탯 정리 -> 마차로 표현
  
### 무기 스탯 정리
  - 무기 24종 관련 스탯
